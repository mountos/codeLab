新しいウェブサイトへようこそ！これは、[Mountos Astro 基本テンプレート：PU 璞](https://pu.code.mountos.com) を使用して構築されたデモページです。この記事は自由に編集または削除できます。

この記事の目的は、Markdown を使用して作成できるさまざまなコンテンツ形式と、このテンプレートの組み込み機能を紹介することです。

## これは H2 見出しです

Markdown を記述することで記事を作成できます。Astro はそれを自動的に HTML に変換します。WYSIWYG で、非常に簡単です。

### これは H3 見出しです

コンテンツを整理するために、異なるレベルの見出しを持つことができます。これは SEO と可読性の両方に非常に役立ちます。

#### リスト

*   これは順序なしリストの項目です
*   多くの項目を持つことができます
    *   ネストされたリストも可能です

1.  これは順序付きリストです
2.  順序は自動的に計算されます
3.  チュートリアルを書くのに最適です

#### 引用ブロック

誰かの言葉を引用したい場合は、引用ブロックを使用できます。

> これは引用ブロックです。テキストの重要性や出典を強調するために、特別なスタイルで表示されます。

#### コードブロック

技術ブログにとって、コードブロックは不可欠です。このテンプレートは、エレガントな構文ハイライトを準備しています。

```javascript
// これは JavaScript コードの例です
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('Astro');
```
